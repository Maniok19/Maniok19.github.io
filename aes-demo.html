What is AES?

The Advanced Encryption Standard (AES) is a symmetric encryption algorithm established by NIST in 2001. It operates on 128-bit blocks of data using keys of 128, 192, or 256 bits. AES is widely used for securing sensitive data in applications ranging from file encryption to secure communications.

AES Algorithm Steps

Step 1

Initial Round

AddRoundKey - XOR the state with the initial round key derived from the cipher key.

Step 2

Main Rounds (9 rounds for 128-bit)

  • SubBytes - Non-linear substitution using S-box
  • ShiftRows - Transposition step with cyclic shifts
  • MixColumns - Linear mixing operation
  • AddRoundKey - XOR with round key
Step 3

Final Round

SubBytes, ShiftRows, and AddRoundKey (MixColumns is omitted in the final round).
